About

This is a web app made for pentesters to test their skills, and practice. There will be multiple vulnarbilities, defense mechanism, and policies in place for you to avoid. We also focus on hand picking a handful of resources for you to learn about the different vulnarbilities. Happy Hunting

This project was made by Ch0ic3. as a way for people to practice their bug bounty skills and improve. I will first focus on getting vulnarbilities up and running, and after that implement real world scenarios. My hope is to include these attacks from bug.bounty.wiki/. This is also not a copy of DVWA but an alternative. It's solely build for personal improvement and insight. Not only that i will strive to make vulnarbilities readable in the server code so peoply can analyse root causes and what practices might be insecure.
